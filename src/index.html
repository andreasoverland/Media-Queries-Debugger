Updates 2011.12.12: The Debugging-tool now supports @mediaqueries inside CSS-files and will scan @imported CSS-files within CSS-files, recursively. Also, a link to the file containing the @mediaquery has been embedded. Try it on The Boston Globe (grab the bookmarklet first)

Copy with pride!
The script loops through all CSS-files containing a "media"-attribute, then tests the media query and reports true or false for a match.

Feel free to copy, alter, enhance and improve! I'll gladly link to any improvements and alternatives to this tool.

How to use
Either drag this Debug MediaQueries bookmarklet onto your toolbar an click on it when you are on the page you want to MQ-debug.
