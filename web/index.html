Updates 2011.12.12: The Debugging-tool now supports @media-queries inside CSS-files and will scan @imported CSS-files recursively. Also, a link to the file containing the @media-query has been embedded. Try it on The Boston Globe (grab the bookmarklet first)

Copy with pride!
The script loops through all external and imported CSS-files and looks for media-queries and tests them to see if they match the current media state.

Feel free to copy, alter, enhance and improve! I'll gladly link to any improvements and alternatives to this tool.

How to use
Either drag this Debug MediaQueries bookmarklet onto your toolbar an click on it when you are on the page you want to MQ-debug.
